Imperial VPN

To connect to the Minecraft server and the Mumble server you need to be on the Imperial network. So if you are not on Imperial's Campus, make sure you are connected to the imperial VPN! If you have not got it setup already you can find all the information on how to set it up here

Minecraft Setup

Installation

  1. Go to the Offical Minecraft Downloads
  2. Download the correct Minecraft launcher for your operating system.
  3. Open the launcher and enter your credentials.

Joining the world without Mumble (Voice chat)

It is important to follow these steps to join the server since the server is run on an older version of Minecraft (1.15.2). Starting the game without selecting the correct version of Minecraft will prohibit you from joining the server.

  1. Open the Minecraft launcher but do not press play.

  2. Click on the Installations tab at the top and create a installation by clicking new.

  3. Choose a name for your installation and in the version tab choose release 1.15.2 from the drop down.
  4. Click on “create” and go back to the play tab.
  5. At the bottom center, select the server you just created and press play.
  6. The launcher will close and a new game window will open.
  7. On the Minecraft home screen select multiplayer.

  8. To add a new server click Direct Connect and enter the server address. It will be in the format IPaddress:port (eg. 123.123.123.123:25565).

  9. Finally connect to the server by clicking Join Server.

Joining the world with Mumble

Follow all the installation sections below then return here once done·

  1. Open the Minecraft launcher, in the version box on the left of the Play button it should say Minercaft forge 1.15.2 if not select this verison from the drop down.
  2. Start the Mumble client and connect to the Mumble server with the right address and port number.
  3. Press Play in the Minecraft launcher and the Minercaft Forge game window will load, it usually takes a little longer than starting an normal verison of minecraft since it is modded.
  4. Once on the home screen you can connect to the server as normal. When you join the Minercaft server in the Mumble client window it should say Minecraft linked.

MumbleLink and Forge

To link Minecraft to Mumble so that we can use 3D positional audio, we need to install Forge a Minecraft modding API and Mumblelink the mod that links Minercaft to Mumble.
