Mumble

To enable users in the Virtual World to communicate to each other through voice chat we use the application Mumble.

Installation

Mumble is well documented making the installation of the server very easy. Mumble Wikipedia

Ubuntu

To install and configure the server use:

sudo apt-get install mumble-server

sudo dpkg-reconfigure mumble-server

Follow the configuring mumble-server prompt and setup the Superuser password. This password is important as it is needed to configure the server.

Configuration

  • To make changes to the config file : sudo (nano/vim) /etc/mumble-server.ini
  • Here you can make changes such as: welcome messages, port, server password, max users. Restart the server to see the changes on the server.
  • To make other changes such as adding channels, ACLs, Admins connect to the server as Superuser via the mumble client app using the correct IP and port of the server.

Server Commands

To start, stop and restart the server

  • sudo service mumble-server start
  • sudo service mumble-server stop
  • sudo service mumble-server restart

Connection as Superuser

The Superuser account is used to configure the initial administrator accounts on the mumble server.

  1. Open the mumble client
  2. Select Connect from the Server menu at the top
  3. Click the Add New button on the bottom of the "Mumble Server Connect" window
  4. Enter a description in the label box. This will help you easily identify the different servers you connect to
  5. Enter the Mumble Serverâ€™s hostname or IP address and port number in the appropriate box
  6. In the Username box enter the name SuperUser
  7. Once you enter the username SuperUser a password box will appear. Make sure you enter the correct password for the user account SuperUser
  8. Press the OK button on the bottom. This will bring you back to the "Mumble Server Connect" window and your newly added connection information will be displayed under the Favorites list
  9. Highlight the server name you would like to connect to and press the Connect button on the bottom of the window

Adding a User to the Admin group

  1. Connect to the server using the account you would like to be made Admin
  2. Register the account by right clicking on your name and selecting register account
  3. Reconnect to the server with the Superuser or Admin account
  4. Right click the Root channel at the top and select edit. The Root channel is where your server properties are stored and any channel below this channel will inherit the account policies unless explicitly excluded
  5. Click on the Groups tab at the top
  6. Select the 'admin' group from the Group dropdown box
  7. Select or Type the username you wish to add to the 'Admin' group in the dropdown box below the Members box. Then click the Add button to the right
  8. Click the OK button on the bottom right
