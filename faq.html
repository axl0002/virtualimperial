FAQ

1. For third party commisioners.

Q: I am interested to use this Imperial virtual world as a platform for my own society events, what steps should I follow?

A: This virtual world is well developed and can be easily adapted for public events that conventionally happens on campus.
To access this vitual world, please have a look at client setup and administration pages. For front end changes and developments, please check world edit / guard pages and contact an admin so we can grant you construction permission and provide related support.

If you would like to hold it on your own server, there might be some administration issues and please contact admin for further details. Technically it is easy to set up a separate copy of this virtual world on another server.
